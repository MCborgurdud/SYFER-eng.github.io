Description

1v1.lol is a fun game where you fight against other players. The goal of the game is to win the match in modes such as 1v1, box fights, and zone wars. If you do not want to go against others you can play an obstacle course, zombie fight, aim train, and rebuild. There are many more fun things to do on 1v1.lol so check it out yourself!
